Display one or more Address on Google map with your icon or your image. All control Google map and display Address when click to icon, image.

== Description ==

TZ Google Map widget you can easy to display multiple address in one map. You can enter your Latitude and Longitude and choose icon or image from local.

<h4>General</h4>
<ul>
<li>Enter your Latitude.</li>
<li>Enter your Longitude.</li>
<li>Choose image from local.</li>
<li>Address: Access HTML code.</li>
<li>Make center: You can check to set this address display center map.</li>
</ul>

<h4>Advanced</h4>
<ul>
<li>Map Height: Enter your height value (default: 450).</li>
<li>Map Color: Color type "hue" and gamma "1".</li>
<li>Map Zoom: Default value (10).</li>
<li>ScrollWheel.</li>
<li>navControl.</li>
<li>MapTypeControl.</li>
<li>ScaleControl.</li>
<li>Draggable.</li>
</ul>

== Installation ==

1. Go to the menu 'Plugins' -> 'Add new' and search for 'TZ Google Map'
2. Click 'install' and active.

== Frequently Asked Questions ==

= How to  add multiple address? =

In "General" tab you can see button "add new" and click it to create new address.

= How to display address in center map? =

Example you have 3 address display in map. You can choose address 1 or address 2 or address 3 display in center.
You go to TZ Google Map widget in "General"  tab you click to address(1) you will see option "Make Center" and checked in it.
